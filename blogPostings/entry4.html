Binary Operations April 13th, 2022 10:20 AM

Def: A binary operation on a non-empty set S is a map B: S x S $\to$ S. "*" is usually used to define a binary operation. Another side note, usually you see the following notation (M, *) where M is the set, and * is the binary operation preformed on that set.

  • Unpacking it:

    • S x S := {(a,b) : a,b$\in$S}.

    S x S $\to$ S

    (a,b) |$\to$ a * b

  • Examples:

    • 1) ($\mathbb{Z}$,*) where a*b=a-b. Find if this operation is communative (a*b = b*a) and associative (a * b) * c = a * (b * c)(will be discussed more next entry).

    Solution: Communative: a - b = b - a, let a = 1, b = 2, then 1 - 2 = 2 - 1 => -1 $\neq$ 1.

    Associative: a - (b - c) = (a - b) - c, let a = 1, b = 2, c = 3, then 1 - (2 - 3) = (1 - 2) - 3 => 2 $\neq$ -2

    Thus I can conclude the binary operation a-b on the integers is not Communative or Associative.

    2) ($\mathbb{R}$, *), where a*b = a+b-ab. Find if this operation is Communative and Associative.

    Solution: Communative: a*b=b*a => a+b-ab = b+a-ba, since we're under normal algebraic structures the following is acceptable. a+b-ab = b+a-ba => -ab = -ba => a = a

    Associative: a * (b * c) = (a * b) * c => (a*b) + c - (a*b) = a + (b*c)-a(b*c) => a+b-ab+c-(a+b-ab)c = a+(b+c-bc)-a(b+c-bc)=>a+b+c-ab-ac-bc+abc=a+b+c-bc-ab-ac+abc, Thus Associative.

    for fun let's check for an inverse: e * a = a * e = e (see this more next node.), It's sufficent to check e * a = a. e * a = a => e + a - ea = a => e - ea = 0. let e = 0, since 0$\in\mathbb{R}$, we can see e -ea = 0 => 0 - 0a = 0 => 0 = 0. Thus an inverse exists, and 0 is the identity element. (e is usually denoted as the identity element in a group).

    Finally ($\mathbb{R}$, *), where a*b = a+b-ab is associative, communative and has an identity/invertible.

Hopefully this is a good intro to binary operations. In the next entry we'll discuss group theory yay!
